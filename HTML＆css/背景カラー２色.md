```sass
$gree: #F3F3F3
$black: #3E3E3E

background: linear-gradient(90deg,$black 0%,$black 980px,$gree 980px,$gree 100%)
```

>方向角度グラデーションの向き

- 0deg　　　 ↑ 　 (下から上へ)
- 90deg　　 → 　 (左から右へ)
- 180deg 　 ↓ 　 (上から下へ)
- 270deg 　 ← 　(右から左へ)

>指定色1　距離

- $black 0%,$black 980px

>指定色２ 距離

- $gree 980px,$gree 100%

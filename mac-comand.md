# 便利コマンド
- cmd + a
全選択
- cmd + d
同じもの選択
- cmd + z
前に戻る
- cmd + k , cmd + 0
インデント全部閉じる
- cmd + k , cmd + j
インデント全部開く
- cmd + n
新しいエディタ開く
- cmd + f
ファイル内の検索
- cmd + sift + f
フォルダ全体を検索
- cmd + option + f
置換

## ちょっとしたテク
- cmd + a,cmd + c,cmd + n,cmd + k,cmd + m
全選択してコピーして、エディタ開いて、言語をHTMLに変更シンタックスハイライトが適用

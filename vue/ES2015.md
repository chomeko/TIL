## functionの省略
```js
new Vue({
  methods: {
    handleclick: function(){ ... }
  }
})
```

>変更


```js
new Vue({
  methods: {
    handleclick() { ... }
  }
})
```

## アロー関数

- map

```js
const newArray = array.map(function(el) {
  return el * 2
})
```

>変更

```
const newArray = array.map(el => { 
  return el * 2
})
```

>さらに省略

```js
const newArray = array.map(el => el * 2)
```

- 複数の引数
```js
const newArray = array.map((el, index) => el * 2)
```


- オブジェクトのreturn

```js
const newArray = array.map( el => ({ value: el * 2 }) )
```
